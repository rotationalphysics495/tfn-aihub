## Key Features

1. **AI Chat Interface** - Natural language queries about plant performance
2. **OEE Dashboard** - Real-time Overall Equipment Effectiveness monitoring
3. **Downtime Analysis** - Pareto analysis of downtime reasons
4. **Safety Monitoring** - Safety incident tracking and alerting
5. **Financial Impact** - Cost-of-loss calculations
6. **Action Engine** - Prioritized daily action recommendations
7. **Memory System** - Conversation history with asset context

## Getting Started

### Prerequisites
- Node.js 18+
- Python 3.11+
- npm 10+

### Quick Start

```bash
# Install dependencies
npm install

# Set up Python environment
cd apps/api
python -m venv venv
source venv/bin/activate
pip install -r requirements.txt

# Start development servers
npm run dev  # Frontend on :3000
uvicorn app.main:app --reload --port 8000  # Backend on :8000
```

See [Development Guide](./development-guide.md) for detailed instructions.
