# TFN AI Hub - Project Overview

## Executive Summary

**TFN AI Hub** is a full-stack Manufacturing Performance Assistant application built as a Turborepo monorepo. It provides plant managers with AI-powered insights into manufacturing performance, including OEE analysis, downtime tracking, safety monitoring, and financial impact assessment.

The application combines a Next.js 14 frontend with a Python FastAPI backend featuring LangChain-based AI agents with 12+ specialized manufacturing tools.

## Architecture Pattern

**Agent-Based Architecture** with:
- **ManufacturingTool** base class for all AI tools
- **ToolRegistry** for automatic tool discovery and registration
- **AgentExecutor** wrapper for LangChain agent orchestration
- **Citation system** for data source tracking (NFR1 compliance)
- **TTL-based caching** for tool response optimization

## Key Features

1. **AI Chat Interface** - Natural language queries about plant performance
2. **OEE Dashboard** - Real-time Overall Equipment Effectiveness monitoring
3. **Downtime Analysis** - Pareto analysis of downtime reasons
4. **Safety Monitoring** - Safety incident tracking and alerting
5. **Financial Impact** - Cost-of-loss calculations
6. **Action Engine** - Prioritized daily action recommendations
7. **Memory System** - Conversation history with asset context
